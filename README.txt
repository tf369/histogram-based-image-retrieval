***************************基于颜色直方图的图片检索***************************
_______________________________________________________________________________________

    1.使用时，先将代码搜索指令路径修改为要搜索的图片库的路径，这里没有将库加入当前目录。
                   
                 list=dir('图片库');

    2.共两种搜素方式，HSV不分区搜索，RGB分区搜索，前者运行时间比后者快，但匹配结果不如后者准确。其余文件为量化（包括均匀和非均匀）函数，求直方图数据函数，以及数据库的统计保存信息，
      示例图片，保存结果图片的文件夹。
    3.使用方法：
       HSV不分区搜索：

                  hsv_imretrieval(im,n); 

       其中，im为模板图，n为返回的图片个数；
       
       RGB分区搜索：
         
                  rgb5_imretrieval(im,n,a1,a2);

       其中，im为模板图，n为返回的图片个数，a1,a2为控制参数，a2越大，搜索结果越偏向原图整体的直方图数据，a1越大，结果更偏向原图各分区的直方图数据。
       a1,a2的建议取值范围[-1 1],超出此区间得到的图像一定是不准确的。
